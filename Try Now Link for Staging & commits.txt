Exercise 1:
Create a new repository named "Git_Exercise_1" with a readme file in the repository, use the following instructions:

1. Clone the link in the Git Bash
2. Add 2 different Java files using the add command
3. Check the status of the directory
4. Commit the files after adding them to the repository
5. Push the files to git using push command.
6. Modify the files by adding a few lines in to the code.
7. Commit them again with a different message.
8. Push the files to git using push command.
9. Check the stash show -p command to see the modifications that are made
10. Check the head command to see the head file directory.

Note: Kindly copy all the code from the Git Bash and paste it here.
